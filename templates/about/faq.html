What is {{ site_name }}?

{% blocktranslate %} {{ site_name }} is a web application for discovering new podcasts and collecting your favorite podcasts and episodes. {% endblocktranslate %}

Is {{ site_name }} free?

{% blocktranslate %} {{ site_name }} is a web application for discovering new podcasts and collecting your favorite podcasts and episodes. {{ site_name }} is free of charge. It's also free, open-source software! {% endblocktranslate %}

Does {{ site_name }} store data about me?

{{ site_name }} keeps track of your podcast subscriptions, bookmarks and individual episodes you listen to. This data is not shared with any third party. If you delete your account, all this data is also permanently deleted. More detailed terms and conditions here.

{% blocktranslate %} Do I have to create an account to use {{ site_name }}? {% endblocktranslate %}

You don't have to sign up to discover new podcasts and episodes and download content.

You'll have to register at {{ site_name }} to do all of these:

  • {% translate "Play episodes in our custom Player" %}
  • {% translate "Bookmark episodes to listen to later" %}
  • {% translate "Add podcasts to your collection" %}
  • {% translate "Get new weekly recommendations" %}
  • {% translate "Keep track of episodes you have listened to" %}
{% translate "What if I want to download an episode to my device?" %}

Click the download icon {% include "icons/download.svg" with css_class="inline h-4 w-4 md:h-6 md:w-6" %} on the episode page to save the audio file to your device. You can then listen to the episode using any suitable audio player software. However if you download on your device, it won't be added to your play History.

{% translate "How do I add podcasts to my collection?" %}

{% blocktranslate %} Just click the Subscribe button on the podcast's About page to add the podcast to your collection. {% endblocktranslate %}

{% blocktranslate %} How does {{ site_name }} recommend new podcasts to me? {% endblocktranslate %}

{% blocktranslate %} We will send you an email with new recommendations every week. We recommend new podcasts using a machine-learning algorithm based on your collection and listening history. {% endblocktranslate %}

{% translate "How do I stop receiving email notifications?" %}

Go to the Preferences section in the Settings area, uncheck the option Send email notifications and click the save button.

{% translate "How do I remove an episode from my history?" %}

Go to the episode page and click the trash icon {% include "icons/trash.svg" with css_class="inline h-4 w-4 md:h-6 md:w-6" %} to permanently remove the episode from your History.

{% translate "Can I import my feeds from another podcast app?" %}

If your podcast app supports OPML export, you can download the OPML file to your device and upload it to {{ request.site.name }} in the Import/Export OPML section in the Settings area.

{% translate "Can I save my podcast subscriptions to my device?" %}

You can download your podcast feed collection in OPML format in the Import/Export OPML section in the Settings area.

How do I add podcasts to {{ site_name }}?

{% blocktranslate %} First try looking for the podcast in the Search box in the navigation bar. If you can't find what you are looking for, click the iTunes link in the search results page. Any new podcasts you find in iTunes will be automatically added to our database. We're also adding new podcasts every day! {% endblocktranslate %}

{% translate "How do I save an episode to listen to later?" %}

Click the star icon {% include "icons/star.svg" with css_class="inline h-4 w-4 md:h-6 md:w-6" %} in the episode page to add an episode to your Bookmarks. You can add episodes you want to listen to later, or favorites you want to listen to again.

{% translate "How often are podcasts updated?" %}

{% blocktranslate %} Popular podcasts are refreshed several times an hour and other podcasts every few hours. Podcasts which haven't submitted new content in a long time may be updated every few days. {% endblocktranslate %}

Which browsers does {{ site_name }} support?

{% blocktranslate %} {{ site_name }} has been tested on Chrome and Firefox browsers on Windows, Linux and Android operating systems. It should work OK on other browsers and operating systems, but no guarantee. If you run into an issue in your particular device or browser please file a bug and provide as much detail as possible. {% endblocktranslate %}

{% blocktranslate %} Absolutely, this is an open source project that can be hosted on your own server. Please see the README for instructions on setting up and running an instance. {% endblocktranslate %}

{% translate "How do I delete my account?" %}

Go to Delete Account section in the Settings area. If you delete your account, your collection, bookmarks and history will also be permanently deleted.
