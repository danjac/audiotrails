Player Shortcuts

These keyboard shortcuts are available when the Player is running.

Key Action
Spacebar Toggle Play/Pause
Left Arrow Skip Backwards 10s
Right Arrow Skip Forwards 10s
+ Increase Playback Rate
- Decrease Playback Rate
0 Reset Default Playback Rate

Navigation Shortcuts

These keyboard shortcuts should be used in combination with Alt+Shift keys on Windows and Linux devices (Chrome/Firefox/Edge) and Ctrl+Option keys on Mac devices (Chrome/Firefox/Safari).

Key Action
p Podcasts
n New Releases
d Discover
q Play Queue
f Favorites
h History
s Settings
a About
k Search
l Login/Logout
j Join
x Close Player (if running)
r Reload Player (if running)
